# Add your headshot here 📸

Drop a photo of yourself in this folder named exactly:

```
me.jpg
```

That's it — the site will pick it up automatically and display it in the
polaroid frame on the homepage. Until then, a fun "JVO" placeholder shows
instead.

**Tips for the best result:**
- Portrait orientation works best (the frame uses a 4:5 ratio)
- Aim for at least 640×800 pixels
- The image is center-cropped, so keep your face near the middle
